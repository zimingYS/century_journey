# CenturyJourney

> Build a living voxel world, one chunk at a time.

一个基于 **Rust** 与 **Bevy Engine** 开发的开放世界体素游戏项目。

CenturyJourney 致力于构建一个拥有丰富生态、动态环境与无限探索空间的体素世界。项目采用模块化架构设计，围绕程序化生成、世界模拟和高性能渲染展开开发。

---

## ✨ 特性

### 🌍 世界生成

* 无限区块世界
* 多层噪声地形生成
* 生物群系系统
* 气候模拟系统
* 季节循环系统

### 🌳 生态环境

* 树木结构生成
* 群系专属植被
* 海洋与沙滩地形
* 动态环境演化（计划中）

### ⚡ 引擎架构

* ECS（Entity Component System）
* 分阶段区块生成流水线
* 跨区块结构生成
* 延迟写入系统
* 存档与加载系统
* 多通道网格构建

### 🎮 游戏内容（开发中）

* 生物系统
* 物品系统
* 建筑系统
* 村庄与遗迹
* 动态天气
* 联机支持

## 🛠 技术栈

| 技术            | 用途     |
| ------------- | ------ |
| Rust          | 核心开发语言 |
| Bevy Engine   | 游戏引擎   |
| ECS           | 游戏逻辑架构 |
| FastNoiseLite | 程序化噪声  |
| Serde         | 数据序列化  |
| RON / JSON    | 配置与存档  |

## 📂 项目结构

```text
src/
├── core/              # 核心配置与状态
├── player/            # 玩家系统
├── voxel/             # 方块注册与属性
├── world/
│   ├── chunk/         # 区块数据
│   ├── generation/    # 世界生成
│   ├── storage/       # 世界存储
│   ├── save/          # 存档系统
│   └── systems/       # ECS 系统
└── main.rs
```
