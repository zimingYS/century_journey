# Kenney sound sources

All sound files in this directory were created and distributed by Kenney and
are licensed under Creative Commons Zero (CC0 1.0). The original license files
are retained in this directory.

Official source packs:

- UI sounds: <https://kenney.nl/assets/ui-audio>
- Interface sounds: <https://kenney.nl/assets/interface-sounds>
- Impact and footsteps: <https://kenney.nl/assets/impact-sounds>
- RPG ambience: <https://kenney.nl/assets/rpg-audio>
- CC0 legal text: <https://creativecommons.org/publicdomain/zero/1.0/>

The files were renamed by function when imported:

- ui: click, rollover, confirmation, error, open and close sounds.
- block: mining, wood, metal and glass impact variants.
- footstep: grass, concrete, wood, snow and carpet variants.
- combat: medium punch impact variants.
- ambient: quiet creak and cloth rustle one-shots.
