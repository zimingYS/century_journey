# Font Licenses

The fonts in this directory are third-party resources and keep their own licenses.

## Fusion Pixel Font 10px Monospaced

The five locale variants under `fusion-pixel/` come from Fusion Pixel Font and are
licensed under the SIL Open Font License 1.1. The Simplified Chinese variant is the
project's default UI font; the Latin, Traditional Chinese, Japanese, and Korean
variants are retained for localization.

The upstream copyright and complete license texts, including notices for bundled
source fonts, are preserved in `fusion-pixel/OFL.txt` and
`fusion-pixel/LICENSES/`.

`NotoSansSC-VariableFont_wght.ttf` is part of Noto Sans SC and is licensed under the SIL Open Font License 1.1.

See the repository-level `THIRD_PARTY_NOTICES.md` for attribution and license details.
